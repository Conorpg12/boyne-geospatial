GIS Data Viewer

GeoPackage Viewer

Upload and inspect GeoPackage (.gpkg) files directly in your browser. View layers, attributes, and geometry - no desktop GIS software needed.

A GeoPackage (.gpkg) is an open, standards-based format for storing and transferring geospatial data - used by GIS professionals working with QGIS, ArcGIS, and field data collection tools. This browser-based viewer lets you inspect the contents of any .gpkg file without installing any software. Simply upload your file to view layers, attributes, and geometry types instantly. Useful for quickly validating data before import, sharing files with non-GIS colleagues, or checking outputs on the go.
